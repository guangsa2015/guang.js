
1、这是一个自创的非MIME的javascript框架，类似于React前端库；<br/>
2、<script type="text/guang">,  guang.view为编译器；<br/>
3、目前实现是<>为求平方函数，render为向body追加element;<br/>
4、代码预览：<br/>
    <script type="text/guang"><br/>
	   const a = 30;<br/>
	   const b = 20;<br/>
	   render(<a>);<br/>
	   render("value:"+<b>);<br/>
	</script><br/>
5、运行效果：<br/>
    900value:400<br/>
